# Oakleaf

<center>

![](./logo.png)

使用Python Fabric库编写的简易自动化运维工具，类似Ansible

![GitHub last commit](https://img.shields.io/github/last-commit/serchaofan/oakleaf)
![Travis (.org)](https://img.shields.io/travis/serchaofan/oakleaf)

</center>

oakleaf目前支持命令
- get
  - hosts
  - groups
- run
  - ping
  - command
  - script
  - sysinfo
  - loadinfo
